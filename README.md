# Прозрачный комфорт

## О проекте

Это сайт-визитка для компании "Прозрачный комфорт", которая занимается установкой мягких окон.

## Как редактировать этот код?

Есть несколько способов редактирования приложения.

**Использование вашей IDE**

Если вы хотите работать локально с помощью своей IDE, вы можете клонировать этот репозиторий и вносить изменения.

Требование: наличие установленных Node.js & npm - [установить с nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Следуйте этим шагам:

```sh
# Шаг 1: Клонируйте репозиторий, используя URL проекта.
git clone <YOUR_GIT_URL>

# Шаг 2: Перейдите в директорию проекта.
cd <YOUR_PROJECT_NAME>

# Шаг 3: Установите необходимые зависимости.
npm i

# Шаг 4: Запустите сервер разработки с авто-перезагрузкой и мгновенным предпросмотром.
npm run dev
```

**Редактирование файла напрямую в GitHub**

- Перейдите к нужным файлам.
- Нажмите кнопку "Edit" (значок карандаша) вверху справа от просмотра файла.
- Внесите изменения и закоммитьте их.

**Использование GitHub Codespaces**

- Перейдите на главную страницу вашего репозитория.
- Нажмите на кнопку "Code" (зеленая кнопка) вверху справа.
- Выберите вкладку "Codespaces".
- Нажмите на "New codespace", чтобы запустить новую среду Codespace.
- Редактируйте файлы напрямую в Codespace и закоммитьте и отправьте изменения, когда закончите.

## Какие технологии используются в этом проекте?

Этот проект построен с использованием:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Как развернуть этот проект?

Для развертывания проекта выполните команду:

```sh
npm run build
```

Файлы для развертывания будут находиться в директории `dist`.
