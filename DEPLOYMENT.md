## Как развернуть сайт

### Способ 1: Через GitHub UI (простой)

1. **Загрузите код в репозиторий**
   ```bash
   git add .
   git commit -m "Добавлены улучшения доступности и функциональности"
   git push origin main
   ```

2. **Активируйте GitHub Pages**
   - Перейдите в репозиторий → Settings → Pages
   - Source: Deploy from a branch
   - Branch: main + / (root)
   - Нажмите Save

3. **Дождитесь сборки** (2-5 минут)
   - Сайт будет доступен по адресу: `https://ваш-логин.github.io/transparent-comfort-landing-main/`

### Способ 2: Через GitHub Actions (рекомендуется)

1. **Создайте файл `.github/workflows/deploy.yml`**:
   ```yaml
   name: Deploy to GitHub Pages
   
   on:
     push:
       branches: [ main ]
   
   jobs:
     build-and-deploy:
       runs-on: ubuntu-latest
       
       steps:
       - name: Checkout
         uses: actions/checkout@v3
         
       - name: Setup Node.js
         uses: actions/setup-node@v3
         with:
           node-version: '18'
           cache: 'npm'
           
       - name: Install dependencies
         run: npm ci
         
       - name: Build
         run: npm run build
         
       - name: Deploy to GitHub Pages
         uses: peaceiris/actions-gh-pages@v3
         with:
           github_token: ${{ secrets.GITHUB_TOKEN }}
           publish_dir: ./dist
   ```

2. **Настройте Permissions**
   - Settings → Actions → General
   - Workflow permissions: Read and write permissions
   - Allow GitHub Actions to create and approve pull requests

## Проверка после развертывания

### Обязательные проверки:
1. **Главная страница**: Загружается ли корректно
2. **Навигация**: Работают ли якорные ссылки
3. **Калькулятор**: Работает ли валидация форм
4. **Мобильная версия**: Адаптивен ли дизайн
5. **404 страница**: Работает ли редирект

### Дополнительные проверки:
1. **SEO**: Проверьте метатеги через [SEO Checker](https://www.seobility.net/en/seocheck/)
2. **Скорость**: Протестируйте через [PageSpeed Insights](https://pagespeed.web.dev/)
3. **Доступность**: Проверьте через [WAVE](https://wave.webaim.org/)

## Возможные проблемы и решения

### Проблема: Белый экран при загрузке
**Решение**: Проверьте `base` в `vite.config.ts` - должен соответствовать пути репозитория

### Проблема: Не работает навигация
**Решение**: Убедитесь что `404.html` находится в корне репозитория

### Проблема: Изображения не грузятся
**Решение**: Проверьте пути к изображениям в компонентах

### Проблема: Сайт не индексируется
**Решение**: Добавьте `robots.txt` и `sitemap.xml`
